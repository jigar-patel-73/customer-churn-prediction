# 📊 Customer Churn Prediction

Author: Jigar Patel  
Aspiring Data Analyst  

---

## 📌 Problem Statement
Predict whether a customer will churn or not using machine learning techniques.

---

## 📊 Dataset
The dataset contains customer demographic and service usage information.

---

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

---

## 📈 Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building (Logistic Regression, Random Forest)
- Handling Class Imbalance using SMOTE

---

## 📊 Results
- Logistic Regression achieved ~81% accuracy  
- SMOTE improved recall for churn prediction significantly  

---

## 🧠 Key Insights
- Customers with low tenure are more likely to churn  
- Higher monthly charges are associated with increased churn  
- Contract type and payment method significantly influence churn  

---

## 🚀 Conclusion
This project demonstrates how machine learning can help businesses proactively retain customers by identifying at-risk customers.
