# 📊 Customer Churn Intelligence

🚀 A Machine Learning web application that predicts whether a customer is likely to churn, helping businesses identify retention risk and improve customer retention strategies.

## 🌐 Live Demo

🔗 https://jigar-customer-churn.streamlit.app

---

## 📌 Project Overview

Customer churn is a major business challenge. Retaining existing customers is often more cost-effective than acquiring new ones.

This project uses Machine Learning to analyze customer attributes and predict churn probability in real time through an interactive web app.

---

## 🎯 Key Features

* Predicts customer churn risk instantly
* Interactive and user-friendly dashboard
* Shows retention probability
* Provides suggested business actions
* Publicly deployed Streamlit web app

---

## 🛠️ Tech Stack

* Python
* pandas
* NumPy
* scikit-learn
* Streamlit
* GitHub

---

## 🤖 Machine Learning Model

* Logistic Regression

Why Logistic Regression?

* Fast and efficient
* Strong baseline classification model
* Interpretable probabilities
* Suitable for churn prediction tasks

---

## 📥 Input Features

Users can enter:

* Senior Citizen
* Customer Tenure
* Monthly Charges
* Contract Type
* Internet Service
* Payment Method

---

## 📈 Output

The app predicts:

* ⚠️ High Churn Risk
  or
* ✅ Strong Retention Probability

Along with probability score and recommended action.
